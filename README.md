
## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` or `yarn` installed globally on your machine.  

Installation:

`npm install` or `yarn install` 

To Run Test Suite:  

`npm test` or `yarn install`  

To Start Dev Server:

`npm dev` or `yarn dev`   

To Visit App:

`http://localhost:5173/`
